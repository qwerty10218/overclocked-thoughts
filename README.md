# ⚡ 思緒超頻 | Gary's Lab (v3.0)

> 「呼吸，是探出水面求生的溫柔。」 — 歡迎來到我的數位萃取實驗室。

這是一個結合了 **React 現代工程**與**精品咖啡美學**的實驗性專案。這裡不只是代碼的堆砌，而是關於一個工程師如何在代碼、爵士樂與手沖咖啡之間尋找平衡的紀錄。

---

## 🛠️ 系統規格 (Technical Specs)

本專案基於 **Vite + React** 構建，並搭載了以下核心引擎：

- **Infinite Virtual Scroll Engine (v3.0)**: 採用虛擬索引技術實現的無限循環輪播系統，確保高效能渲染。
- **Framer Motion**: 驅動全站的物理感交互與「呼吸感」動畫。
- **Tailwind CSS**: 打造極簡的賽博龐克（Cyber-Coffee）視覺風格。
- **Lucide Icons**: 精準的戰術圖標系統。

## 🧪 核心概念 (Core Concepts)

1. **KERNEL SOURCE (單一產區)**: 追蹤風味的源頭，如同追蹤原始碼。
2. **PRECISION BUILD (精密萃取)**: 恆定壓力的數位編譯，不容 1bit 誤差。
3. **SYSTEM RESET (數位排毒)**: 離線是為了下一次更強大的連線。
